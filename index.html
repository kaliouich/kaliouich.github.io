n8n

n8n est une plateforme d'automatisation de flux de travail. L'objectif de cette application est de vous permettre de connecter différentes applications et API entre elles afin d'automatiser vos tâches de manière fluide.

Cette instance utilise l'authentification OAuth pour interagir de manière sécurisée avec divers services tiers (comme Google) et exécuter les automatisations que vous définissez.
